1. Noter / Evaluer un vin
Sur cette page figurent tous les vins de la base de données liée à l'application. Si vous avez déjà noté ce vin, alors il sera affiché avec sa note :
Dans le cas où vous n'avez pas encore noté le vin, alors il est affiché avec une note qui correspond à la moyenne des notes des concours pour ce vin suivi, entre parenthèses, du nombre de notes constituant la moyenne :

Si la note est bonne ou excellente (4 ou 5), elle est affichée en vert.
Si la note est moyenne (3), elle est affichée en orange.
Si la note est médiocre (2), elle est affichée en rouge.

Pour noter un vin, il suffit de toucher la petite fiche (voir images ci-dessus).
2. Choisir un millésime
Une fois le vin sélectionné, la page suivante permet de choisir un millésime (année spécifique dans une tranche de 1990 à 2019) pour votre bouteille dans une liste déroulante.


Une fois le millésime sélectionné, touchez "Valider" pour passer à l'écran suivant.
3. Fiche du vin
Sur la fiche du vin vous pouvez consulter les informations détaillées du vin et du millésime sélectionné. Ce qui est intéressant ici en revanche, c'est la possibilité de pouvoir évaluer le vin avec une note et un commentaire (facultatif) :

Un touché sur le bouton "Annuler" vous ramène à l'écran de sélection des vins tandis ce que "Enregistrer l'avis" sauvegarde votre note et votre commentaire dans la base de données et affiche une fenêtre de confirmation avec deux options :
Une fois le vin noté, vous avez le choix soit de retourner à la sélection des vins (continuer à noter), soit de consulter votre historique personnel.
4. Mon historique personnel
Sur cette page vous retrouvez une liste des vins que vous avez évalués auparavant, filtrés par catégorie (rouge, blanc, rosé et mousseux) et par ordre alphabétique.
Dès que vous touchez un en-tête (vins blancs par exemple), celui-ci s'ouvre et laisse place aux notes des vins sous forme de ♥ (coeurs). Les coeurs que vous voyez sont en fait une moyenne de tous les millésimes du vin. Pour modifier votre note / noter un nouveau millésime, cliquez sur le vin de votre choix afin d'être redirigé vers la page de sélection du millésime.

5. Trouver un restaurant
Cette page vous permet de retrouver les restaurants à proximité afin de pouvoir consulter leur carte des vins.
Une carte plein écran vous permet de retrouver les restaurants répertoriés dans la base de données du système. Ces derniers sont indiqués sur la carte avec des marqueurs rouges tandis que la position de l'utilisateur qui consulte cette page est en bleue.
Un touché sur un marqueur rouge affiche une bulle info avec les données du restaurant sélectionné. Touchez "Afficher la carte des vins" afin d'accéder à la carte du restaurant souhaité :
Cette dernière ressemble visuellement à la page de mon historique personnel mais à la différence affiche les pourcentages de recommandation pour les vins susceptibles de vous plaire.
